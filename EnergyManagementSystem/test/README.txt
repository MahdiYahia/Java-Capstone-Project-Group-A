How to run the test? 


Step-by-Step Instructions for Eclipse:
Install JUnit (if not already installed):

Right-click on the project and choose "Build Path" > "Add Libraries...".
Select JUnit from the list and ensure it's the correct version (JUnit 5).
Navigate to the Test Class:

Go to the src/test/java/energy/ directory and open the test class you want to run (e.g., ChargingStationTest.java).
Run the Test:

Right-click on the test class or method.
Select "Run As > JUnit Test".
The results will be shown in the JUnit view.
